AI-powered Forest Fire Risk Prediction Dashboard using Machine Learning and Real-Time Weather Data
# EcoShield AI 🔥

## Intelligent Forest Fire Risk Prediction System

EcoShield AI is a Machine Learning-based forest fire prediction dashboard developed using Python and Streamlit.

The system predicts forest fire risk using:

* Temperature
* Humidity
* Wind Speed
* Real-time Weather Data

## Features

* Real-time weather monitoring
* Forest fire risk prediction
* Interactive dashboard
* Emergency fire alerts
* Forest monitoring map
* Machine Learning model integration

## Technologies Used

* Python
* Streamlit
* Scikit-learn
* Pandas
* Matplotlib
* Open-Meteo API

## Machine Learning Algorithm

* Random Forest Classifier
